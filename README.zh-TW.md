## 用法💨

GitHub[發布](https://github.com/HidemaruOwO/Puyodeliver/releases)或行動[人工製品](https://github.com/HidemaruOwO/Puyodeliver/actions/workflows/build.yml)請從中下載可執行檔並運行它。

```bash
mkdir public
PUBLIC_FOLDER=public/ ./puyodeliver
```
